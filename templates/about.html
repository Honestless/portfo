03 : About me

About Sebastian:

I was born and raised in Romania. When I was 21 years old I decided to move to Czech Republic,

to find a better job and I did. I was testing different electronics for Amazon which was really

an amazing job, especially for being able to try out the latest gadgets and technologies.

From 0 to Mastery

In 2020 I found a udemy course by Andrei Neagoie, which is called from 0 to Mastery which caught my eye. After watching some of the videos I realised how fun programming can be so I finished some of the course beginner part to get some basics and started working on my own projects. Whenever I would get stuck a friend which works in the field would help out, while still spending time watching video courses to extend my knowledge.

Where am I so far

Since I started learning I tried to stick with games, only for the fact that I needed to keep myself focused and interested in order to overcome all the errors and bugs. My biggest project so far is a game called Unholy: A chosen hero, which I'm developing since November. Since the time I started learning, I got more than basics in pyGame, BeautifulSoup and also got the chance to work with databases such as MySQL and SQlite, and during the time I started building up this website, I also got to work with Flask Framework. I learned some HTML, CSS and javascript, just enough to edit the template I'm using in order to make it more personal.
